# Part_1

Данный раздел содержит отчет об изучении предметной области, выявлении сущностей и их атрибутов, проектировании схемы базы данных, создании и заполнении базы данных. Также в разделе привдены примеры запросов к данной БД.

1. Отчет о выполненной работы.pdf - отчет составленный в рамках оразовательного курса. Данный отчет содержит: 
   1.1 Описание предметной олбасти
   1.2 Определение целей функционирования БД, ролей, сущностей и атрибутов
   1.3 Составление  ER-диаграммы и схемы сущностей 
   1.4 Проектирование схемы БД и описание таблиц 
   1.5 SQL скрипт создания БД
   1.6 Python код  генерации данных для заполнения БД
   1.7 Примеры SQL запросов к БД и разбор их работы

2. All_sql_request.sql - sql скрипт со всем запросами, представленными в отчете о выполнении работы
3. Creat_dog_base_v1.sql -sql скрипт создания БД
4. ERD.pdf -  ER-диаграмма по описанию предметной олбасти.
5. SBDA.pdf - Спроектированая схема БД на английском языке
