# DB_Dog_Monitoring

Данный репозиторий содержит материалы по разработке и использованию базы данных для мониторинга информации о бездомных и домашних собаках. База данных разрабатывалась в качестве учебного проекта в рамках образовательных дисциплин "Теоретические основы баз данных" и "Методы проектирования баз данных"

## Part_1

Данный раздел содержит отчет об изучении предметной области, выявлении сущностей и их атрибутов, проектировании схемы базы данных,  создании и заполнении базы данных. Также в разделе привдены примеры запросов к данной БД.

## Part_2

Данный раздел содержит отчет о работе с разработанной в Part_1 БД с помощью представлений, триггеров, функций, процедур и транзакций.
