    1.1 Создание представления и его использование в запросах к БД.
    1.2 Создание и применение триггеров к БД
    1.3 Создание пользователей с разными правами и сравнение действий данных пользователей
    1.4 Создание функций для работы с данными БД
    1.5 Изучение аномалии <<грязное чтение>> на уровне транзакции с уровнем изоляции READ COMMITED
2. Shem.pdf -  схема БД с добавлением новой таблицы для проверки работы тригеров
3. zapros_for_view_v1.sql - sql скрипт для работы спредставлением
4. zapros_func.sql - sql скрипт для создания и применения функций
5. zapros_tranzaction.sql - sql скритп для работы с транзакциями
6. zapros_trigger.sql - sql скритп для работы с триггерами
